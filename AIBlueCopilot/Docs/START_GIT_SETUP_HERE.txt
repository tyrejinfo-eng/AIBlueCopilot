## ?? Key Concepts Explained

### What is a Branch?
Independent copy of code where you work without affecting others.

### What is a Commit?
Save point with description. Like "checkpoint" in a video game.

### What is a Pull Request?
"Please review my code before merging" - asks team for approval.

### What is a Merge?
Combining your branch changes back into main/develop branch.

### Why Do We Do This?
```
? Without Git:
- No backup
- Can't collaborate
- Lose work easily
- No code history

? With Git:
- Automatic backups ?
- Easy collaboration ?
- Work preserved ?
- Complete history ?
```
