## ?? Key Concepts

```
BRANCH: Independent copy of code. You work in feature/*, 
        team merges to develop, released from main.

COMMIT: Save point with description. Like "Checkpoint" 
        in a video game.

PUSH: Upload your commits to GitHub so team can see them.

PULL: Download latest code from GitHub so you're in sync.

PULL REQUEST (PR): "I have code, please review before merge"
                   Team reviews, approves, then merge.

MERGE: Combine your branch into another branch 
       (usually develop or main).
```
